# PhishGuard Privacy Policy

Last updated: 2026-03-31

## Summary
PhishGuard is a browser extension that helps users detect phishing websites, spam emails, and suspicious text. The extension performs checks locally and, when AI analysis is used, sends the content being analyzed to OpenRouter for processing.

## Data we handle
PhishGuard may process the following data to provide its core functionality:

- Websites visited (URL/domain) for website safety analysis.
- Website content that you explicitly select and analyze via the right‑click menu.
- Text you paste into the extension popup for analysis.
- Gmail email content (subject, sender, and email body text) when you open an email in Gmail, to show an in‑page safety banner.

## Data storage
PhishGuard stores the minimum necessary data locally in Chrome extension storage:

- Your OpenRouter API key (if you configure it).
- Cached analysis results for the current tab/session.
- Your feedback (for example, when you report a site as safe/phishing).

PhishGuard does not run analytics and does not create advertising identifiers.

## Data sharing / third parties
PhishGuard may transmit data to the following third parties strictly to provide the extension’s single purpose (phishing/spam detection):

- **OpenRouter**: When AI analysis runs, the extension sends the URL or text being analyzed to OpenRouter’s API and receives an analysis result.
- **OpenPhish**: The extension downloads the OpenPhish feed (a list of known phishing URLs) to perform threat-intel checks. This download does not include user data.

PhishGuard does not sell user data.

## Remote code
PhishGuard does not execute remote JavaScript or WebAssembly code. All extension code is packaged with the extension.

## Your choices
- You can remove the extension at any time.
- You can clear extension storage via Chrome’s extension settings.
- You can clear your OpenRouter API key in the extension Options page.

## Contact
If you need support or have privacy questions, contact the publisher using the support contact listed on the Chrome Web Store listing.
